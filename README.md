# 🤖 AI Sentiment Analyzer
## Live APP:
https://ai-sentiment-analysis-k5wffgovfxvj4s8cfa8tcs.streamlit.app/
An advanced NLP-based Sentiment Analysis web application built using Machine Learning and Streamlit.

## 🚀 Features

- Real-Time Sentiment Detection
- Positive / Negative / Neutral Prediction
- Beautiful Modern UI
- Confidence Score Meter
- NLP Text Cleaning
- TF-IDF Vectorization
- Logistic Regression Model
- Streamlit Web App

## 🛠 Technologies Used

- Python
- NLP
- Scikit-learn
- Streamlit
- TF-IDF
- Logistic Regression

## 📂 Dataset

Sentiment140 Twitter Dataset
